# Promotion Checklist

Use this checklist when moving work from experiment branch to production branch.

## Product validation
- Hypothesis was explicit and testable
- Evidence shows meaningful user value signal
- Decision rubric score recorded

## Engineering quality
- Unit tests added for core logic
- Integration tests cover key path
- Regression risk reviewed
- Error handling verified

## Operational readiness
- Feature flag in place
- Telemetry and alerts added
- SLO impact reviewed
- Rollback process tested

## Security and compliance
- Secrets handling reviewed
- Authz/authn impacts reviewed
- Data classification and retention reviewed
- Required security scans passing

## Delivery controls
- CI checks green
- Code review complete
- Release notes updated
- Ownership on-call confirmed

## Final decision
- Promote now
- Promote with staged rollout
- Block and remediate
